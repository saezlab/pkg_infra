# About

`pkg_infra` is a shared infrastructure package for the Saezlab Python
ecosystem. It was created to replace ad hoc runtime support code with a small,
maintainable layer built on top of standard Python tooling.

The project focuses on three recurring needs across packages:

- Logging
- Configuration handling
- Session metadata

The broader project context, design motivation, and intended use cases are
described in the pages within this section.
